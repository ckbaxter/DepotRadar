Parqet Integration
ℹ️ Wie erstelle ich eine Parqet Integration?
Einmalige Einrichtung bei Parqet:
  1. Gehe zu developer.parqet.com/console und klicke auf + New Integration
  2. Name: beliebig, z.B. ATH-Tracker
  3. Scopes: nur read portfolio ankreuzen — write portfolio weglassen
  4. Redirect URIs: folgende URL eintragen (exakt so, inkl. /api/parqet/callback): 💡 Tippen → Alles auswählen → Kopieren
  5. Create klicken → erscheinende Client ID kopieren und oben einfügen
